# Lecteur MP3 Local Android

Projet Android natif en Kotlin pour lire des musiques MP3 stockees en local sur le telephone.

## Fonctions incluses

- selection de plusieurs fichiers audio avec le selecteur Android
- playlist locale
- lecture / pause
- piste precedente / suivante
- barre de progression
- memorisation de la playlist choisie

## Ouverture du projet

1. Ouvre le dossier `C:\Users\vince\Downloads\LocalMp3PlayerAndroid` dans Android Studio.
2. Lance la synchronisation Gradle.
3. Demarre l'application sur un telephone Android ou un emulateur.

## Remarque

Le wrapper Gradle n'a pas ete genere ici, car `gradle` n'est pas installe dans cet environnement. Android Studio peut ouvrir le projet et proposer la synchronisation automatiquement.
